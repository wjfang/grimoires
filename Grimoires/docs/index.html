Grimoires 1.7.0 Documentation

  • Summary

    • Introduction
      What Grimoires is? What Grimoires does?
    • Release Notes
      Gives a short overview of this Grimoires release: the summary of supported functionalities, deployment environments, prerequisite software, etc.

  • Getting Started

    • Getting Started Guide
      Gives a very simple guide to install and use Grimoires.

  • Installation & Use

    • Installation Guide - Grimoires 
      Provides a detailed guide on how to configure Grimoires, how to compile and make Grimoires binaries from source code, and how to deploy Grimoires binaries to the target container.
    • Installation Guide - GShell 
      Describes how to compile and run GShell.
    • User Guide - GShell
      GShell is a command line client tool for Grimoires. It provides a shell and simple interactive commands to publish/inquire business/service/wsdl/metadata. These commands can also be called in a batch mode outside the shell.
    • Installation Guide - UDDI Browser
      Describes how to install UDDI Browser, and how to configure UDDI Browser so that it supports signing SOAP messages using X509 certificate managed by the OMII client.
    • User Guide - UDDI Browser
      UDDI Browser is a UDDI client GUI that can be used to interact with Grimoires' UDDIv2 interface.
    • HOWTO - Lifetime Management
      Describes how to make use of the lifetime management provided by the Grimoires registry.
    • HOWTO - Query for TriplesAssertion Metadata
      Describes how to query for TriplesAssertion metadata by using a RDF query language, RDQL.
    • HOWTO - Direct RDF Query against Grimoires
      Describes how to execute a RDF query against Grimoires through its newly introduced RDF interface.
    • HOWTO - GrimoiresProxy
      Describes GrimoiresProxy: a convenient client side library.
    • Publication and Discovery of Feta Description
      Describes the way to publish and discover Feta description using the Grimoires registry.
    • XMLView
      Describes the XMLView interface that supports XPath query and XSL transformation between application service description schema and Grimoires schema.

  • Architecture

    • Implementation Design - Grimoires

    • Functional Specification - GShell
      Describes the functionality and the use of GShell from the user's perspective.
    • Implementation Design - GShell

  • Support

    • Tutorial - Grimoires Service
      Describes how to interact with Grimoires web services by sending and receiving SOAP messages.
    • Tutorial - Grimoires Client Side Java API
      Demonstrates how to write a Grimoires client program using Grimoires Java API.
    • Tutorial - GShell Client
      Describes three ways to use GShell: an interactive mode, a batch mode, and sending soap message directly.
    • Tutorial - GShell Secure Client
      Demonstrates how GShell would interact with a secure Grimoires deployment
    • Tutorial - Client Side UDDI4J Java API
      UDDI4J is an open source UDDI client API that can be used to interact with Grimoires' UDDIv2 interface. UDDI4J's library and documentation are released with Grimoires. UDDI Browser relies on UDDI4J.
    • JavaDoc
      Grimoires API JavaDoc.
